## Setup

- Clone o repositório e instale dependências:

```bash
git clone https://github.com/DenylsonMiguel/HistoryCalc.git
cd HistoryCalc
npm install
```

- Crie o arquivo .env na raiz do projeto com:

```
PORT=<port>
```

- Rode em desenvolvimento:

```bash
npm run dev
```
